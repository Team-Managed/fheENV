# Web Dashboard

The fheENV dashboard at [fheenv.vercel.app](https://fheenv.vercel.app) provides a wallet-connected
view of projects and their public on-chain evidence.

## Features

- **Create projects** on-chain with one click
- **List owned projects** from the registry
- **Review on-chain evidence** for rotations and access changes
- **Export public on-chain events** as CSV

Secret push, pull, rotation, and team access are currently CLI workflows. The dashboard does not
read decrypted values, wallet keyfiles, or private keys.

## Connect Wallet

Click "Connect Wallet" and approve with MetaMask, Coinbase Wallet, or any injected EVM wallet. The
dashboard operates on **Ethereum Sepolia** testnet.

If you're on the wrong network, the app will prompt you to switch.

## Project List

After connecting, you'll see projects where your wallet is an owner. A project page shows registry
metadata and public events indexed from Sepolia. The CSV includes only data emitted on-chain plus
the transaction sender indexed by Blockscout; it does not infer local `pull` or `run` activity, old
CIDs, or IPFS unpinning.
