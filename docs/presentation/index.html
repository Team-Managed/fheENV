Why FHE is the only way.

ZK proves you know a secret — it can't selectively share it. AES needs a trusted key server.

FHE stores the AES key encrypted on-chain. Access is granted cryptographically — to wallet addresses only.

fheENVRegistry.sol
// 1. Store AES-256 key as FHE ciphertext
euint128 aesKeyHigh = FHE.asEuint128(inHigh);
euint128 aesKeyLow = FHE.asEuint128(inLow);

// 2. Grant teammate access on-chain
FHE.allow(aesKeyHigh, teammateAddress);

// 3. Revoke access instantly
FHE.revokeAllow(aesKeyHigh, formerMember);
